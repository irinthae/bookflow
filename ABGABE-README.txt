# HOW-TO README

## Script

  Das CMD-Script komprimiert Ihr Projekt in ein ZIP-Archiv und 
  überspringt dabei nicht relevante Dateien und Ordner, um die
  Größe der Abgabe möglichst klein zu halten.


## Abgabe durchführen

- Kopieren Sie alle Ordner und Dateien in Ihr Projektverzeichnis:
	IHR-PROJEKTVERZEICHNIS/
	|- lib/
	|- .zipignore
	|- abgabe.cmd
	|- ... (ihre Projektdaten)
	
	
- Öffnen Sie die Datei: abgabe.cmd
	- Ersetzen Sie den String MUSTERMANN mit Ihrem Zunamen (BLOCKBUCHSTABEN)
	- Ersetzen Sie den String Max mit Ihrem Vornamen

- Führen Sie das Script abgabe.cmd aus
	- Nach dem Archivieren sollte sich automatisch ein neues Explorer-Fenster öffnen,
	  in dem sich das ZIP-Archiv IHRZUNAME_Vorname-mein-projektname befindet.
	  


## Ubload Teams

- Geben Sie Ihre Datei in Teams unter Aufgaben `Abgabe Vaadin Projekt` ab.
